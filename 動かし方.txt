run.pyがあるディレクトリで
  python run.py
を実行

【出来なかった時】
  pip install -r requestments.txt
を実行（足りないライブラリのインストールをしている）

【初期設定】
git remote add origin https://github.com/iikyara/iStand.git
git push -u origin master
を一回ずつ実行しておく

【ダウンロード】
git pull origin master

【アップロード】
git add .
git commit -am "comment"
git push
の3手順でアップロード
